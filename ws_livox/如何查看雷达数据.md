# 如何查看Livox雷达数据

恭喜！您的雷达已经成功连接，可以看到以下话题：
- `/livox/lidar` - 点云数据
- `/livox/imu` - IMU数据

## 方法1: 使用rostopic命令查看（最简单）

### 查看点云数据
```bash
# 实时打印点云数据（会显示很多数据）
rostopic echo /livox/lidar

# 只查看消息头信息（减少输出）
rostopic echo /livox/lidar | grep -E "header|point_num|timebase"

# 查看发布频率
rostopic hz /livox/lidar
```

### 查看IMU数据
```bash
# 实时打印IMU数据
rostopic echo /livox/imu

# 查看发布频率
rostopic hz /livox/imu
```

### 查看话题信息
```bash
# 查看话题类型
rostopic type /livox/lidar
rostopic type /livox/imu

# 查看话题详细信息
rostopic info /livox/lidar
rostopic info /livox/imu
```

## 方法2: 使用RViz可视化（推荐，最直观）

### 启动带RViz的驱动
```bash
cd ~/ws_livox
source devel/setup.bash
roslaunch livox_ros_driver2 rviz_MID360.launch
```

### 或者单独启动RViz
如果驱动已经在运行，在另一个终端：
```bash
cd ~/ws_livox
source devel/setup.bash
rviz
```

### 在RViz中配置：
1. **Fixed Frame**: 设置为 `livox_frame`
2. **Add** → **By topic** → 选择 `/livox/lidar` → **PointCloud2** 或 **CustomMsg**
3. 调整显示设置（点大小、颜色等）

## 方法3: 使用Python脚本查看

使用之前创建的脚本：
```bash
cd ~/ws_livox
source devel/setup.bash
python3 read_lidar_data.py
```

脚本会自动订阅 `/livox/lidar` 话题并打印点云信息。

## 方法4: 使用rosbag录制和回放

### 录制数据
```bash
# 录制所有话题
rosbag record -a

# 或只录制雷达相关话题
rosbag record /livox/lidar /livox/imu

# 录制到指定文件
rosbag record -O lidar_data.bag /livox/lidar /livox/imu
```

### 回放数据
```bash
# 回放bag文件
rosbag play lidar_data.bag

# 回放时查看数据
rosbag play lidar_data.bag &
rostopic echo /livox/lidar
```

## 方法5: 查看数据统计信息

```bash
# 查看点云数据统计
rostopic echo /livox/lidar -n 1 | grep point_num

# 查看消息大小
rostopic bw /livox/lidar

# 查看消息详细信息
rostopic echo /livox/lidar -n 1
```

## 数据格式说明

### CustomMsg格式（xfer_format=1时）
```bash
rostopic echo /livox/lidar
```
会显示：
- `header`: ROS消息头
- `timebase`: 时间基准
- `point_num`: 点的数量
- `lidar_id`: 雷达ID
- `points[]`: 点数组，每个点包含：
  - `x, y, z`: 3D坐标
  - `reflectivity`: 反射率
  - `tag`: 点标签
  - `line`: 扫描线编号

### PointCloud2格式（xfer_format=0时）
```bash
rostopic echo /livox/lidar
```
会显示标准的sensor_msgs/PointCloud2格式。

## 快速检查清单

✅ **检查数据是否正常**:
```bash
# 1. 检查话题是否存在
rostopic list | grep livox

# 2. 检查发布频率（应该接近10Hz）
rostopic hz /livox/lidar

# 3. 查看一条消息
rostopic echo /livox/lidar -n 1
```

## 常见问题

### Q: rostopic echo输出太多怎么办？
A: 使用过滤：
```bash
# 只查看点数量
rostopic echo /livox/lidar | grep point_num

# 只查看前几个点
rostopic echo /livox/lidar -n 1 | head -50
```

### Q: 如何保存数据用于后续分析？
A: 使用rosbag录制：
```bash
rosbag record -O my_lidar_data.bag /livox/lidar /livox/imu
```

### Q: 如何可视化点云？
A: 使用RViz（最推荐）：
```bash
roslaunch livox_ros_driver2 rviz_MID360.launch
```
