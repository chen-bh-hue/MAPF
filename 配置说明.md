# Livox雷达驱动配置指南

## 快速开始

在新机器上配置Livox雷达驱动，只需运行以下命令：
sudo ip addr add 192.168.1.100/24 dev eth0
```bash
cd /home/bingda
./setup_livox_driver.sh
```

脚本会自动完成以下步骤：
1. ✅ 检查ROS环境
2. ✅ 安装缺失的依赖包
3. ✅ 编译安装Livox-SDK2
4. ✅ 配置网络IP地址
5. ✅ 编译ROS工作空间
6. ✅ 配置环境变量

## 手动配置步骤（如果脚本失败）

### 1. 安装依赖

```bash
sudo apt update
sudo apt install -y cmake libpcl-dev libapr1-dev
sudo apt install -y ros-noetic-pcl-ros  # ROS1
# 或
sudo apt install -y ros-humble-pcl-conversions  # ROS2
```

### 2. 编译安装Livox-SDK2

```bash
cd /home/bingda/Livox-SDK2
mkdir -p build && cd build
cmake ..
make -j$(nproc)
sudo make install
```

### 3. 配置网络IP

编辑配置文件：
```bash
nano /home/bingda/ws_livox/src/livox_ros_driver2/config/MID360_config.json
```

确保 `host_ip` 与主机实际IP地址匹配：
```json
{
  "host_ip": "192.168.1.100"  // 改为你的主机IP
}
```

查看主机IP：
```bash
ip addr show | grep "inet " | grep -v "127.0.0.1"
```

### 4. 编译ROS工作空间

**ROS1 (Noetic):**
```bash
cd /home/bingda/ws_livox
source /opt/ros/noetic/setup.bash
catkin_make -DROS_EDITION=ROS1
source devel/setup.bash
```

**ROS2 (Humble/Foxy):**
```bash
cd /home/bingda/ws_livox
source /opt/ros/humble/setup.bash  # 或 foxy
colcon build --cmake-args -DROS_EDITION=ROS2
source install/setup.bash
```

### 5. 配置环境变量

将以下内容添加到 `~/.bashrc`：

**ROS1:**
```bash
source /home/bingda/ws_livox/devel/setup.bash
```

**ROS2:**
```bash
source /home/bingda/ws_livox/install/setup.bash
```

然后运行：
```bash
source ~/.bashrc
```

## 启动驱动

### ROS1
```bash
cd /home/bingda/ws_livox
source devel/setup.bash
roslaunch livox_ros_driver2 msg_MID360.launch
```

### ROS2
```bash
cd /home/bingda/ws_livox
source install/setup.bash
ros2 launch livox_ros_driver2 msg_MID360_launch.py
```

```bash
rospack find livox_ros_driver2
```

**ROS2:**

**ROS1:**
```bash
rostopic list | grep livox
```

**ROS2:**
```bash
ros2 topic list | grep livox
```

## 常见问题

### 1. 编译错误：找不到livox_lidar_sdk库
**解决：** 确保Livox-SDK2已正确安装
```bash
sudo make install  # 在Livox-SDK2/build目录下
```

### 2. IP地址不匹配
**解决：** 修改配置文件中的 `host_ip` 或配置网络接口
```bash
sudo ip addr add 192.168.1.100/24 dev eth0
```

### 3. 话题不出现
**原因：** 话题是延迟创建的，需要等待雷达连接
**解决：** 等待5-30秒，确保雷达已连接并上电

### 4. 权限问题
**解决：** 确保有sudo权限，或使用sudo运行安装命令

## 网络配置

雷达默认IP: `192.168.1.157`
主机IP需要配置为: `192.168.1.100` (或修改配置文件)

配置网络接口：
```bash
sudo ip addr add 192.168.1.100/24 dev eth0
# sudo modprobe spidev
sudo ip link set eth0 up
```

测试连接：
```bash
ping 192.168.1.157
```
